## üöÄ Quick Commands

### Run Locally:
```bash
# Terminal 1: Backend
cd management
docker-compose up -d postgres ollama
mvn spring-boot:run

# Terminal 2: Frontend
cd frontend
npm start
```

### Deploy:
```bash
# Frontend
cd frontend
npm run build
# Deploy to Vercel

# Backend
cd management
# Deploy to Railway
```
